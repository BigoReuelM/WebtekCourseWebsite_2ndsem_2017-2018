Hypertext Markup Language(HTML)

Hypertext Markup Language (HTML) is used for creating web pages and web applications for everyone to use anywhere. Proposed by physicist Sir Tim Berners-lee in 1989 during his stay at CERN, it has since inproved and become the core language in creating content for the web.

HTML Version Timeline

  • 1991 HTML 1.0

    • A hybrid of SGML with a text markup language with the ability to link documents.
    • Get simple text up on the web.
    • 22 elements.

  • 1995 HTML 2.0

  • HTML 3.0 draft

    • Included several new abilities and more powerful opportunities for webpage developers to create their pages.
    • Browsers were slow in implementing these new abilities and opportunities.
    • HTML 3.0 draft was abandoned.

  • In 1997, HTML 3.2 became the official standard.

  • 1998 HTMML 4.0 (Cougar) to HTML 4.01

    • Published as a recommendation in late 1997.
    • Cascading Style Sheets (CSS) was included.
    • Became the official standard in April of 1998.
    • Later revised to be HTML 4.01.

  • XHTML 1.0 and XHTML 1.1

    • XHTML is a stricter, cleaner and more efficient vertion of HTML.
    • Became a standard in January 2000 together with HTML 4.01.

  • HTML 5

    • 2014 W3C Recommendation.
    • 2012 HTML 5.1 draft, 2016 W3C Recommendation.
    • 2015 HTML 5.2 Draft, 2018 W3C Recommendation.
